# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## About

CodeKAI is a collection of programming exercises presented as numbered Markdown files (001–021+), ordered by increasing difficulty. They cover basic programming structures and are designed for coaching sessions where an experienced programmer guides a learner, or for self-study when learning a new language.

## Repository Structure

- Each exercise is a single `.md` file named `NNN-short-name.md` containing the problem statement
- No source code, build system, or tests — this is a problem-set repo, not a codebase
- Exercises are language-agnostic; the learner picks any language

## Conventions

- Files are numbered sequentially with zero-padded 3-digit prefixes (e.g., `013-rot13.md`)
- When adding a new exercise, use the next available number and a short kebab-case name
- Exercise files contain only the problem description (plain text + optional code blocks for expected output)
